# Myntra-functional-clone
1.Developed a functional clone of Myntra using HTML, CSS, and JavaScript, replicating key features like product listings, filtering, and a responsive design.
2.Designed an intuitive and visually appealing user interface with CSS for a seamless shopping experience. Implemented interactive features such as product search, category filtering, and a shopping cart using JavaScript. 
3.Optimized the website for responsiveness and cross-browser compatibility to ensure a smooth user experience on different devices.
4.Enhanced UI/UX by adding hover effects, animations, and a structured layout for easy navigation.
